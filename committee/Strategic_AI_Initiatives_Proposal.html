AI Technical Curriculum Development

Creating Structured Learning Pathways for High-Demand AI Skills

üéØ Strategic Objective

Develop comprehensive, industry-aligned curriculum teaching modern AI development skills (prompt engineering, RAG systems, agentic AI) that 8% of Austin employers currently require (growing 32.8% annually through 2030) but ACC currently doesn't teach. Establish prompt engineering as a foundational competency that can serve as a general education requirement while providing technical depth for CSIT students.

Initiative 2.1: Prompt Engineering Fundamentals (Course 1)

Foundational course teaching systematic prompt design, chain-of-thought reasoning, few-shot learning, and prompt testing methodologies. Accessible to all majors as potential gen ed competency (AI literacy), while providing depth for technical students. 3 credit hours.

üìñ Level: Beginner (no prerequisites)
üéØ Skills: Prompt design, testing, documentation
üë• Audience: All majors + CSIT technical track

Initiative 2.2: RAG Systems & Document Intelligence (Course 2)

Intermediate course covering vector databases, semantic search, document chunking, embeddings, and retrieval-augmented generation architectures. Students build production RAG applications using institutional tool access. Prerequisite: Course 1 or programming experience.

üìñ Level: Intermediate (requires prompt eng or coding)
üéØ Skills: Vector search, embeddings, RAG architecture
üèóÔ∏è Project: Build production chatbot or Q&A system

Initiative 2.3: Agentic AI & Multi-Step Systems (Course 3)

Advanced course teaching autonomous AI agents, tool use, multi-step reasoning, state machines, and complex workflows. Students build agents that can use external tools, make decisions, and execute multi-step plans. Capstone-level complexity. Prerequisite: Courses 1 & 2.

üìñ Level: Advanced (requires Courses 1 & 2)
üéØ Skills: Agent design, tool integration, orchestration
üöÄ Outcome: Portfolio-ready autonomous AI system

Initiative 2.4: Curriculum Integration & Faculty Development

Integrate AI curriculum into existing programs: add to CSIT degree plans, offer as certificate, position Course 1 as general education competency. Provide faculty professional development on teaching prompt engineering and AI systems. Partner with TLED for training infrastructure.

üë®‚Äçüè´ Faculty PD: Summer workshops, ongoing support
üìö Integration: CS, Software Dev, LAN Admin programs
